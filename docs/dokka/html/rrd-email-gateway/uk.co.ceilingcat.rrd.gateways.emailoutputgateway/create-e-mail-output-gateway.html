createEMailOutputGateway

fun createEMailOutputGateway(emailUserName: EmailUserName, emailPassword: EmailPassword, emailFrom: EmailFrom, emailTo: EmailTo, emailBodyText: EmailBodyText, subjectTemplate: SubjectTemplate): UpcomingOutputGateway
Content copied to clipboard

Construct a new email output gateway.

Mails are sent using SMTP, using a hardcoded provider.

The subjectTemplate permits the replacement of a single token of the value <<serviceType>>'; it is replaced with: 'Refuse'when the service type is ServiceType.REFUSEand, 'Recycling', when the service is of type ServiceType.RECYCLING`.

Parameters

emailUserName

the SMTP user's name

emailPassword

the SMTP user's password

emailFrom

the email's source mailbox name

emailTo

the email's destination mailbox name

emailBodyText

the email's body text

subjectTemplate

the email's subject template.
